# My Portfolio

A personal portfolio website of frontend developer Oleksandra Skybina, built with Next.js.
The project presents basic information about me, my skills, contact details, and a dedicated page with my projects.

The website includes project descriptions: each project contains a title, preview image, link to the live version, and a short description explaining the implemented features and the technologies used.

## Features

- Home page with a hero section and key skills.
- About page with information about me.
- Projects page with project cards and descriptions.
- Contact page with links to email, GitHub, and LinkedIn.
- Responsive interface and navigation between sections.

## Tech Stack

- `Next.js 16`
- `React 19`
- `TypeScript`
- `CSS Modules`
- `Keen Slider`

## Project Structure

- app/ — application pages and routes
- components/ — reusable UI components
- data/ — project data used on the portfolio page
- public/ — images, icons, and static files
- types/ — TypeScript types and skill lists
